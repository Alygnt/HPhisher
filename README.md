## INSTALLATION :
  
### Termux
* `pkg update -y`
* `pkg upgrade -y`
* `pkg install git -y`
* `git clone https://github.com/Alygnt/HPhisher`
* `cd HPhisher`
* `bash hphisher.sh`

### Kali Linux
* `sudo apt update -y`
* `sudo apt upgrade -y`
* `sudo apt install git -y`
* `git clone https://github.com/Alygnt/HPhisher`
* `cd HPhisher`
* `sudo bash hphisher.sh`

### Google cloud shell
